# Marketing-Campaign-A-B-Testing-Analysis

# Project Background

**Marketing Campaign A/B Testing Analysis**

An online platform wants to understand  whether showing users **advertisements** increases product conversions.

Users were randomly split into two groups:

  **1.    Ad Group** – users saw advertisements

1. **PSA Group** – users saw a Public Service Announcement (PSA), which acts as a neutral control message

The goal was to determine if advertisements lead to **higher conversion rate**


# Dataset Overview

**Dataset**

The dataset contains **~580,000 user observations** from a marketing campaign experiment.

Each record includes:

1. user_id – unique user identifier
2. test_group – ad (treatment) or psa (control)
3. converted – whether the user purchased the product
4. total_ads – number of ads shown to the user
5. most_ads_day – day with highest ad exposure
6. most_ads_hour – hour with highest ad exposure
